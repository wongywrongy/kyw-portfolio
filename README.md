# Portfolio Website

A personal portfolio website built as a monorepo with a Next.js frontend and Sanity CMS backend.

## Tech Stack

**Frontend**
- Next.js 16
- React 19
- Tailwind CSS
- shadcn/ui + Radix UI

**Backend**
- Sanity v3 (Headless CMS)

**Deployment**
- Vercel

## Structure

```
frontend/    → Next.js website
backend/     → Sanity Studio (CMS)
```
